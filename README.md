## 📋 Proje Hakkında

SalesVisionAI, büyük ölçekli e-ticaret verileri üzerinde veri analizi, yapay zeka ve makine öğrenmesi tekniklerini bir araya getiren kapsamlı bir satış analizi ve tahminleme platformudur.

Proje kapsamında 2 milyondan fazla sipariş verisi analiz edilerek, müşteri davranışları, satış trendleri ve gelir tahminleri üzerine içgörüler elde edilmiştir. Geliştirilen sistem sayesinde işletmeler, geçmiş verileri analiz ederek geleceğe yönelik daha doğru ve stratejik kararlar alabilmektedir.

Uygulama içerisinde RFM analizi ile müşteri segmentasyonu yapılmakta, ML.NET kullanılarak müşteri sadakat skorları tahmin edilmekte ve farklı ülke ile şehirler için satış tahminleme modelleri oluşturulmaktadır. Ayrıca OpenAI ve HuggingFace API entegrasyonları sayesinde müşteri yorumları analiz edilmekte, duygu analizi (sentiment analysis) ve dil çevirisi işlemleri otomatik olarak gerçekleştirilmektedir.

Modern ve kullanıcı dostu dashboard yapısı sayesinde tüm veriler gerçek zamanlı olarak görselleştirilmekte, karmaşık veri yapıları anlaşılır ve interaktif grafikler ile sunulmaktadır.

Bu proje, gerçek dünya verisiyle büyük veri analizi, yapay zeka entegrasyonları ve makine öğrenmesi uygulamalarının bir dashboard platformunda nasıl kullanılabileceğini gösteren kapsamlı bir eğitim projesidir.

## 🛠️ Kullanılan Teknolojiler

### 🔧 Backend
- ASP.NET Core 8.0 MVC
- Entity Framework Core
- SQL Server (2+ milyon satır veri analizi)
- ML.NET (Makine Öğrenmesi)

### 🎨 Frontend (Web UI)
- HTML5
- CSS3
- JavaScript
- Bootstrap
- jQuery
- Leaflet.js - Coğrafi harita görselleştirme
- Chart.js - İnteraktif grafikler ve line/doughnut chartlar

### 📊 Veri Analizi & Görselleştirme
- RFM Analizi (Recency, Frequency, Monetary)
- Dashboard KPI'lar ve istatistikler
- Ülke bazlı satış tahminleri (Forecasting)
- Ödeme yöntemi tahminleri
- Almanya şehirleri sipariş tahminleri
- Real-time sipariş trend grafikleri

### 🤖 Yapay Zeka & Machine Learning

- OpenAI GPT-4 - Müşteri yorum analizi ve satın alma davranışı değerlendirmesi
- HuggingFace API - Dil çevirisi ve sentiment analiz modelleri
- ML.NET - Customer Loyalty Score tahmini (İtalya müşterileri için sadakat skoru)
- Sentiment Analysis (Pozitif/Negatif/Nötr duygu analizi)

### 📈 Uygulama Özellikleri
- Müşteri segmentasyonu
- Coğrafi satış dağılımı (harita)
- Stok takibi ve uyarı sistemi
- Ürün / kategori / sipariş yönetimi
